How it works

1

Connect slots

Click Create API key, copy the key from Google, then paste it into Slot management below and click Save slots. Elgoog tracks slot readiness and uses failover when quota is exhausted.

2

Bring context

Paste a repo summary, docs excerpt, TODO list, or unfinished-work note. You can also point at a file or repo, or drop a local file into the input area.

3

Run a work task

Choose a concrete task like recover work, create TODOs, or understand repo. The app maps that to a Gemini-first task class.

4

Inspect artifacts

Every run writes an auditable artifact with slot, model, status, and output so the system stays trustworthy.
